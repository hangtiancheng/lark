This is a demo app for [react-vue](https://github.com/antfu/react-vue) library. react-vue makes Vue Compostion API and Vue plugins work in React apps.

## How to use

```bash
npx degit antfu/react-vue/examples/react-ts-vite
cd react-ts-vite
yarn install
yarn dev
```
